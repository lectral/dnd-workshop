# Economy Appraiser Precedents

Use this file to store first-time price estimates for items not found in `shop_1.csv`.

## Rules

- Add one heading per unique item.
- Update an existing entry instead of duplicating the same item.
- Keep rationale short and based on analogs from `shop_1.csv`.
- Use ISO date format `YYYY-MM-DD`.

## Entry Template

## <Item name>
- Date: YYYY-MM-DD
- Estimated Price: <value in GP>
- Range: <low-high in GP>
- Analogs: <item 1>, <item 2>, <item 3>
- Context: <settlement/shop/rarity assumptions>
- Rationale: <1-3 concise lines>
