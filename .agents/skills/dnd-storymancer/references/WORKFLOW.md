# Storymancer Workflow

## Purpose
Break predictable brainstorming by forcing inspiration through a random textual shard.

## Recommended pattern

1. Run `node scripts/pick-fragment.js --format json`.
2. Note the fragment, source book, and emotional texture.
3. Map it to one or more of these creative levers:
   - image
   - conflict
   - secret
   - omen
   - location mood
   - title or epithet
4. Generate the requested result.
5. End with a short `Inspiracja:` note explaining the fragment's influence.

## Good uses

- quest hooks
- NPC concepts
- villain titles
- relic names
- prophecies
- eerie tavern rumors
- one-shot premises
- symbolic scene dressing

## Bad uses

- exact quotations longer than a short fragment
- pretending the fragment is canon lore
- forcing a result that ignores the user request
